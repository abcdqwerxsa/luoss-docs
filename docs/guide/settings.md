# 设置

设置页面用于管理您的账户信息、安全设置、SSH 公钥和查看审计记录。

## 功能概述

| 功能 | 说明 |
|------|------|
| **个人信息** | 查看个人信息和账户状态 |
| **安全设置** | 修改登录密码 |
| **SSH 公钥** | 管理 SSH 公钥，用于开发环境和调试节点访问 |
| **审计记录** | 查看账户操作审计日志 |
| **系统配置** | 训练任务运行时长限制（仅管理员可见） |

## 个人信息

### 查看信息

| 字段 | 说明 |
|------|------|
| 用户名 | 登录用户名（不可修改） |
| 邮箱 | 注册邮箱 |
| 用户 ID | 系统分配的唯一标识 |
| 注册时间 | 账户创建时间 |
| 账户权限 | 平台管理员 / 普通用户 |
| 账户状态 | 已激活 |

## 安全设置

### 修改密码

1. 切换到 **安全设置** 标签页
2. 输入当前密码
3. 输入新密码（至少 6 个字符）
4. 确认新密码
5. 点击 **确认**

::: warning 安全提示
- 使用强密码
- 定期更换密码
- 不要与他人共享密码
:::

## SSH 公钥管理

SSH 公钥用于通过 SSH 连接 [开发环境](/guide/environments) 和调试节点。上传公钥后，您可以使用本地 IDE（VS Code、PyCharm 等）远程连接到您的云端环境。

### 生成 SSH 密钥对

如果您还没有 SSH 密钥，请在本地终端中生成：

::: code-group

```bash [Ed25519（推荐）]
ssh-keygen -t ed25519 -C "your_email@example.com"
```

```bash [RSA]
ssh-keygen -t rsa -b 4096 -C "your_email@example.com"
```

:::

生成后，密钥对位于：
- **私钥**：`~/.ssh/id_ed25519`（或 `id_rsa`）— 请妥善保管，不要泄露
- **公钥**：`~/.ssh/id_ed25519.pub`（或 `id_rsa.pub`）— 需要上传到平台

### 添加公钥

1. 切换到 **SSH 公钥** 标签页
2. 复制公钥内容：

```bash
# 在本地终端查看公钥
cat ~/.ssh/id_ed25519.pub
```

3. 将公钥内容粘贴到输入框中
4. 点击 **添加**

::: tip 提示
- 支持添加多个公钥（例如不同的设备使用不同的密钥）
- 公钥格式通常为：`ssh-ed25519 AAAA... user@host`
- 添加后立即生效，无需重启环境
:::

### 管理公钥

已添加的公钥会显示在列表中：

| 操作 | 说明 |
|------|------|
| 查看 | 显示公钥指纹（用于验证） |
| 删除 | 移除不再使用的公钥 |

::: danger 删除公钥
删除公钥后，使用该密钥的 SSH 连接将立即失效。请确保保留至少一个有效公钥。
:::

### 连接测试

添加公钥后，可以测试 SSH 连接：

```bash
# 测试连接（使用环境详情页面中显示的连接命令）
ssh -p <端口> root@<服务器地址>

# 成功后会看到类似输出：
# Welcome to Ubuntu ...
# Last login: ...
```

### 常见问题

#### Q: 提示 Permission denied？

- 确认已上传正确的公钥（检查公钥内容是否完整）
- 确认使用的私钥与上传的公钥配对
- 检查私钥文件权限：`chmod 600 ~/.ssh/id_ed25519`

#### Q: 提示 Host key verification failed？

服务器密钥可能已更新，执行：

```bash
ssh-keygen -R "[<服务器地址>]:<端口>"
```

然后重新连接。

#### Q: 可以上传多个公钥吗？

可以。每个公钥对应一个设备或环境，建议为不同设备使用不同的密钥对。

## 审计记录

审计记录标签页显示您账户的所有操作日志。

### 支持的操作类型

| 操作 | 说明 |
|------|------|
| 登录 / 登出 | 账户登录和登出记录 |
| 创建环境 | 创建开发环境 |
| 启动 / 停止环境 | 环境启停操作 |
| 删除环境 | 删除开发环境 |
| 创建训练任务 | 提交训练任务 |
| 启动 / 停止训练 | 训练任务启停 |
| 删除训练任务 | 删除训练任务 |
| 修改密码 | 密码变更记录 |
| 添加 / 删除 SSH 公钥 | SSH 密钥管理操作 |

### 筛选功能

- **操作类型**：按操作类型筛选
- **状态**：成功 / 失败
- **时间范围**：选择日期和时间区间

### 审计字段

| 字段 | 说明 |
|------|------|
| 时间 | 操作发生的时间 |
| 操作 | 执行的操作类型 |
| 详情 | 操作的详细信息 |
| IP 地址 | 操作来源 IP |
| 状态 | 成功或失败 |

## 系统配置（仅管理员）

管理员可以在系统配置标签页设置训练任务的全局限制：

| 配置项 | 说明 | 范围 |
|--------|------|------|
| 全局最大运行时长 | 训练任务的最大运行时间（分钟） | 1 - 10080 分钟（1-7 天） |
| 超时检查间隔 | 系统检查超时任务的时间间隔 | 30 - 3600 秒 |
| 自动终止 | 是否自动终止超时的训练任务 | 开 / 关 |

## 常见问题

### Q: 忘记密码怎么办？

A: 联系管理员重置密码。

### Q: 用户名可以修改吗？

A: 用户名创建后不可修改。

### Q: SSH 公钥和密码有什么区别？

A: SSH 公钥用于通过 SSH 客户端连接开发环境和调试节点；密码用于登录平台 Web 界面。两者相互独立。
