# 常见问题

## SSH 连接相关

### Q: SSH 连接提示 Permission denied？

**A**: 请检查以下内容：
1. 确认已在 [设置 > SSH 公钥](/guide/settings#ssh-公钥管理) 中上传公钥
2. 确认使用的私钥与上传的公钥是同一对
3. 检查私钥文件权限：`chmod 600 ~/.ssh/id_ed25519`
4. 确认环境处于 Running 状态

### Q: SSH 连接提示 Host key verification failed？

**A**: 服务器密钥可能已更新（例如环境重建后）。执行以下命令清除旧密钥：

```bash
ssh-keygen -R "[<服务器地址>]:<端口>"
```

然后重新连接，接受新的 host key。

### Q: SSH 连接后 conda 命令不可用？

**A**: conda 环境通常会在 SSH 登录后自动加载。如果没有加载：

```bash
# 手动加载
source /root/.bashrc

# 或重新初始化
eval "$(/root/miniconda3/bin/conda shell.bash hook)"
```

### Q: 如何在 VS Code 中连接开发环境？

**A**: 安装 Remote-SSH 扩展后，按 `Ctrl+Shift+P`，选择 `Remote-SSH: Open Configuration File...`，添加：

```
Host luoss-env
    HostName <服务器地址>
    Port <端口>
    User root
    IdentityFile ~/.ssh/id_ed25519
```

详细步骤请参考 [快速开始](/guide/getting-started#使用-vs-code-remote-ssh)。

## 环境相关

### Q: 环境启动失败怎么办？

**A**: 检查以下内容：
1. 配额是否充足（查看仪表盘）
2. 资源配置是否在允许范围内
3. 查看环境详情的错误信息
4. 联系管理员获取帮助

### Q: 环境自动停止了？

**A**: 系统会在环境空闲超过设定时间后自动停止。如需继续使用：
1. 点击 **启动** 按钮重新启动
2. 考虑增加自动停止时间
3. 定期访问环境更新访问时间

### Q: 删除环境后数据还在吗？

**A**: 
- `/models` 共享存储中的数据**会保留**（跨环境共享）
- 容器 overlay 层的数据**默认会被清除**（除非删除时勾选"保留数据"）
- 建议重要文件存放在 `/mnt/model` 或 `/models` 目录

### Q: 环境一直处于 Creating 状态？

**A**: 
1. 等待 2-3 分钟（overlay 首次创建需要格式化 256G 存储镜像）
2. 如果超过 10 分钟仍未就绪，检查节点资源是否充足
3. 查看环境事件或联系管理员

## 训练任务相关

### Q: 任务一直处于 Pending 状态？

**A**: 可能原因：
1. **集群资源不足** - 等待其他任务释放资源
2. **优先级较低** - 被高优先级任务抢占
3. **NPU 资源不足** - 检查 NPU 配额和使用情况

解决方案：
- 降低资源配置
- 提高任务优先级
- 等待资源释放

### Q: 训练任务失败如何排查？

**A**:
1. 进入任务详情页
2. 查看 **事件** 了解失败原因
3. 查看 **日志** 检查运行日志
4. 常见错误：
   - OOM（内存不足）：增加内存配置
   - 镜像拉取失败：检查镜像地址
   - 存储挂载失败：检查存储配额

### Q: 如何查看训练输出？

**A**: 训练输出保存在 `/models/output` 目录：
1. 通过 SSH 连接到开发环境查看
2. 或通过数据存储功能下载

## 存储相关

### Q: 存储空间不足怎么办？

**A**:
1. 清理不需要的文件
2. 删除旧的模型检查点
3. 联系管理员增加配额

### Q: 如何在不同环境间共享文件？

**A**: 所有环境共享同一个用户存储（`/models`），直接访问对应路径即可。

### Q: 如何备份数据？

**A**:
```bash
# 通过 SSH 连接后打包重要文件
cd /models
tar -czvf backup.tar.gz projects/

# 或使用 SCP 下载到本地
scp -P <端口> -r root@<服务器地址>:/models/projects/ ./
```

## SSH 公钥相关

### Q: 支持哪些 SSH 密钥类型？

**A**: 支持 Ed25519（推荐）和 RSA（至少 4096 位）。

### Q: 可以上传多个公钥吗？

**A**: 可以。建议为不同设备使用不同的密钥对，方便管理。

### Q: 删除公钥后正在进行的 SSH 连接会断开吗？

**A**: 删除公钥后，新的 SSH 连接将无法建立，但已建立的连接不会立即断开。

## 配额相关

### Q: 如何查看我的配额？

**A**:
1. 查看仪表盘的配额概览
2. 进入 **使用统计** 页面查看详情

### Q: 配额不足怎么办？

**A**:
1. 停止不使用的环境
2. 删除已完成的训练任务
3. 清理存储空间
4. 联系管理员申请增加配额

### Q: NPU 配额为什么是 0？

**A**: 默认用户可能没有 NPU 配额。请联系管理员分配。

## 账户相关

### Q: 忘记密码怎么办？

**A**: 联系管理员重置密码。

### Q: 如何修改密码？

**A**:
1. 点击左侧菜单 **设置**
2. 切换到 **安全设置** 标签页
3. 输入旧密码和新密码

## 其他问题

### Q: SSH 到调试节点可以跑训练任务吗？

**A**: **不可以。** 通过 SSH 登录调试节点（裸机）后，仅供代码编辑、文件管理和环境调试使用，严禁直接在裸机上执行计算任务。但通过平台创建的 [开发环境](/guide/environments) 和 [训练任务](/guide/training-jobs) 可正常执行各类计算任务，不受此限制。

### Q: 项目文件应该存放在哪里？

**A**: **项目文件请优先存储到 `/mnt/model` 目录下。** 该目录为大容量共享存储，适合存放代码、数据集、模型等。`/models` 为用户主目录，也可用于存放个人项目。

### Q: 平台支持哪些开发模板？

**A**: 平台提供多种预配置模板：

| 模板 | 说明 |
|------|------|
| MindIE | 华为昇腾推理开发 |
| MindSpeed | LLM 训练和推理 |
| vLLM | 高性能 LLM 推理 |
| MindSpore | MindSpore 框架开发 |

### Q: 如何联系技术支持？

**A**:
- 联系平台管理员
- 查看系统公告获取联系方式

### Q: 如何报告 Bug？

**A**:
1. 记录问题发生的时间和操作步骤
2. 截取错误信息截图
3. 联系管理员
